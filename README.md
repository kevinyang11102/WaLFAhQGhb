# Python货物进销管理系统 python76

## 项目概述

本项目是一款基于Python的货物进销管理系统，集成了采购订单管理、商品信息管理、供应商管理、销售管理、仓库管理、成员管理以及个人信息修改等主要功能模块。

## 技术栈

- 后端：Python + Flask
- 前端：HTML + CSS + JavaScript + jQuery
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0

## 功能模块

- 登录注册
- 采购订单管理
- 商品信息管理
- 供应商管理
- 销售管理
- 仓库管理
- 成员管理
- 个人信息修改
